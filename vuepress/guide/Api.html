# Devnagri Platform APIs

# Login API

This API enables client to login

POST: https://app.devnagri.com/api/login

# Request

{
    "email": "test.client@devnagri.com",
    "password": "****"
}

# Response

{
    "token": "asdiasdoiasdubdebdsbdaysbduaysbduaysbd"
}

# Generate API Token

POST https://app.devnagri.com/api/key/validations

# Request

{
    "client_id": "id-of-client",
    "client_secret": "client-secret",
    "project_key": "project-key"
}

# Response

{
    "token": "Your-token"
}

Put this token in all headers while sending the request to devnagri.

Example:

Accept: 'Application/json'
Authorization: 'bearer your-token-here'

# JSON order using API

POST: https://app.devnagri.com/api/order/json

# Request

{
    "api_key": "SECRET_API_KEY_XXXXX",
    "project_name": "My Sample Order",
    "industry": "sports",
    "source_language": "en",
    "target_language": [
        "hi",
        "bn"
    ],
    "data": [
        {
            "id": 1,
            "OriginalText": "Hello World"
        },
        {
            "id": 2,
            "OriginalText": "How are you?"
        }
    ]
}

# Response

{
    "status": "success",
    "code": 200,
    "message": "Order placed successfully.",
    "order_id": "DEV00510"
}

# Notes

  1. This API is responsible to create order without platform interface using a secret API key.
  2. After getting success response from the API, the user can then login to the platform (opens new window), and see the order.

# Request Parameters

Key Value Description
api_key SECRET_API_KEY_XXXXX The key will be provided by the Devnagri and can be found at Usage Details (opens new window)
project_name My Sample Order Name of the order
industry other Type of industry/domain the order belongs to. Ref.
source_language en Language of the order which needs to be translated from. This will be a single value. Ref.
target_language [hi, bn] Language of the order which needs to be translated to. This will be inside an array, can contains more than one language. Ref.
data Your JSON data A valid JSON object
